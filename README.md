# CodeSavant-AI

## 🚀 Project Overview
CodeSavant-AI is an AI-powered **code review** platform that leverages **Google Gemini AI** to analyze and provide insightful feedback on code. Built using the **MERN stack**, this project enables developers to get AI-generated code reviews without storing data in a database. Instead, it directly interacts with APIs to process and return results in real-time.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite)  
- **Backend:** Node.js, Express.js  
- **AI Integration:** Google Gemini AI  
- **API-Based Data Flow** (No database storage)  

## ✨ Features
- **AI-powered Code Review** using Gemini AI.
- **Real-time analysis** via API (No data stored in MongoDB or any DB).
- **User-friendly interface** built with React.js.
- **Efficient Backend** using Express.js and Node.js.

## 🏗️ Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Aanandshukla02/CodeSavant-AI.git
cd CodeSavant-AI
```

### 2️⃣ Install Dependencies
#### Backend:
```bash
cd backend
npm install
```

#### Frontend:
```bash
cd frontend
npm install
```

### 3️⃣ Run the Project
#### Start Backend Server
```bash
cd backend
npx nodemon
```

#### Start Frontend
```bash
cd frontend
npm run dev
```

## 📌 Usage
1. **Enter your code snippet** in the input field.
2. **Submit for review**, and the AI will analyze the code.
3. **Receive AI-generated feedback** instantly.
